# HeatWaves

A tool for analyzing fire occurrence data from the BRASK (Norwegian fire statistics database).

## Features

- Extract data from BRASK website
- Clean and preprocess fire data
- Perform comprehensive analytics (trends, patterns, risks)
- Generate visualizations and reports
- Interactive dashboard for data exploration

## Getting Started

1. Clone this repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the extractor: `python extract_data.py`
4. Run the analysis: `python analyze_data.py`
5. Launch the dashboard: `python fire_analytics_dashboard.py`
